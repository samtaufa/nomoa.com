Introduction

The following pages are just a collection of sections within IMP that can be readily modified, customised. The customisability of IMP is important to the project team (http://www.horde.org) so to get the latest information log into their site and download the current release.

The Cover Page

[ref: ../imp/config/MOTD.html]

Login Prompt. This is a plain text file that is 'echo'd' onto the HTML login page. The text displays below the login prompt, and there is a lot of space for writing.

HTML tags can be used in this file to beautify your message.

I find this a good location to place the E-mail policy statement for the organisation.

[ref: ../imp/config/menu.txt]

Left Margin Links. This is a configuration text file that allows you specify additional items to be placed in the left-margin column.

This column normally has only the IMP logo and the "login" button. The link will cause a new browser window to be started.

Graphics files are expected to be in the ../imp/graphics directory.

[ref: ../imp/config/trailer.txt]

Trailer Messages. The contents of this file are echoed to the mail sent by IMP clients.

PHP variables are expanded before messages are sent.

[ref: ../imp/locale/en/login.lang]

contains standard English text for the login screen. Let's you change the 'welcome to IMP' to another welcome message.

To make this a global change (independent of language) then make the change to ../imp/locale/login.lang. My suggestion is to copy the language file you would normally use into this folder, change the sections you want to keep and remove the other sections.

[ref: ../imp/locale/en/status.lang]

Top Toolbar. Contains the text for the top toolbar 'Welcome to IMP. Please log in.' Changing this effects the text in the message:

To make this a global change (independent of language) then make the change to ../imp/locale/status.lang. My suggestion is to copy the language file you would normally use into this folder, change the sections you want to keep and remove the other sections.

Page Title

The page title is defined in ../imp/index.php3. The <title> is defined twice, once for browsers with support for frames, and another for unknown browsers.
