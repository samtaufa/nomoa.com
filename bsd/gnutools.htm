Colorising ls

[ref: ls colorized misc@openbsd.org October 2001]

The standard file listing tool 'ls' in OpenBSD does not support colorising output. To support this, one method is to install the GNU ls.

# cd /usr/ports/misc/fileutils

# make install

Edit your script ".profile" to include something like the below.

alias ls="/usr/local/bin/gls -pb --color=always"

and then reload the profile by using

$ source ~/.profile
