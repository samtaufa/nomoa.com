Introduction

DHCPd documentation is rather straight forward and these notes are to document some of the things that don't seem to be clearly documented (or in a quick to find location.)

Related files to configuring the DHCPd server.

/etc/dhcpd.conf - configuration file
/var/db/dhcpd.leases - maintains existing leases in case of server shutdown

/etc/rc.conf.local - configuration file to enable dhcpd between system startups

/etc/dhcpd.interfaces - list of interfaces to which dhcpd will send ip

Documentation can be found in many different places (refer the man pages configuration files above and dhcpd man pages) as well as in the FAQ.


shared-network LOCAL-NET {
    option  domain-name "colorado.edu";
    option  domain-name-servers 128.138.129.76, 128.138.240.1;

    subnet 10.0.0.0 netmask 255.255.255.0 {     
                option routers 10.0.0.1;
                default-lease-time 3600;

                range 10.0.0.33 10.0.0.34;     
                range 10.0.0.38 10.0.0.127;

                host judith {     
                     hardware ethernet 00:90:27:66:de:a5;     
                     fixed-address 10.0.0.32;     
                }
                host baelzebub {     
                     hardware ethernet 0:d0:9:2c:be:6e;     
                     fixed-address 10.0.0.35;     
                }     
   }     
}
