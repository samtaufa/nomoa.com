Staff Training

One serious, and continuing problem, with development of Computer Studies and computing in Tonga has been the oversight and disregard for existing staff. The school systems stand to gain significantly when their staff are empowered by the knowledge available through IT and the empowerment for improved quality service.

The following are a list of proposed minimal skills for effective, independent computing by staff members. (listed in no particular order)

  • File Management
  • Creating Notes
  • Incorporating graphics into notes
  • Creating Examination Papers
  • Collating Records
  • Analysing Records
  • Data Security practises (backing up, anti-virus, passwords)

The goal is increased skills to lead to increase usage to lead to increased efficiencies and increased facilities for students.

Technical Staff Skills

With the increased dependence on IT, the technical staff will need to recognise the need for their edification in matters such as: (again in no particular order)

  • 1st Level Operational Skills
    1. Administer Accounts (create, modify, delete)
    2. Administer Groups (create, modify, delete)
    3. Access Control Lists (use, change)
    4. Policy Systems
    5. Password / User Authentication
    6. Security Policies
  • 2nd Level Operational / Systems Maintenance
    1. Install and Remove user applications
    2. Install and Remove the Operating System
    3. Install and Remove network devices
    4. Diagnose and evaluate hardware problems
    5. Diagnose and evaluate software problems
  • 3rd Level Operational / Optimization
    1. Develop scripts for automated administration
    2. Develop procedures to minimize workload

Note that the above list of skills are for the smooth, optimal function of the computer facilities and in no way describes the needs of the courses operated by the schools.
